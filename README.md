# Figma Weave · Image Model Field Guide

A practical guide to every image model available in Figma Weave — what each one does, when to reach for it, and how designers actually think about the landscape.

Covers text-to-image generation, image editing & compositing, vector/SVG, upscaling, 3D, and specialist/LoRA tools, plus a quick-reference cheat sheet.
